Girl Develop It

We created a new site that is dynamic, responsive, and scalable as the organization continues to expand. Our redesign puts an emphasis on clarity and user experience, with a style that reflects the caliber and importance of the work that the organization does. The site has been rebuilt from the ground up, keeping all the original content in place and adding new features such as a database and API to manage chapter information, a map that makes it easy to find a local chapter, an administrative interface for national and chapter leaders, and a streamlined organization of the existing content.

Girl Develop It is a nonprofit organization that provides affordable programs for adult women interested in learning web and software development in a judgment-free environment.
